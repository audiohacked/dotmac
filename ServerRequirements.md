# Introduction #

This document details the server operating system and software requirements for dotmac.


# Details #

## Operating Systems ##

A unix or linux based operating system is highly recommened. All development and guides are written assuming deployment on linux or unix.

### Recommened Operating Systems ###
  * Redhat
  * Ubuntu
  * CentOS
  * Debian

## Software ##

  * Apache HTTPd Webserver
  * mod\_perl
  * mod\_auth
  * mod\_dav
